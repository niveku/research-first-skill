# Contributing

Contributions are welcome. This skill improves when real-world workflows surface gaps or better resources.

The skill is built for Claude Code but is compatible with any agent that supports SKILL.md-based skills (Codex, Gemini CLI, Cursor, and others). Contributions that improve compatibility across agents are especially welcome.

## Ways to contribute

**Improve existing content:**
- Better search queries for a domain
- More accurate domain-fit examples
- Fixes to outdated resource links or tool names

**Extend to new domains:**
- The skill currently covers dev, writing, data, and ops — other domains (legal, design, finance, etc.) are fair game if the research-first principle applies

**Report issues:**
- Skill triggering when it shouldn't (or not triggering when it should)
- Recommended resources that are abandoned, biased, or no longer relevant

## How to submit changes

1. Fork the repo
2. Edit `research-first/SKILL.md`
3. Open a PR with a short description of what you changed and why

No special setup needed — the skill is a single Markdown file.

## Guidelines

- Keep the skill focused: it should remain a *process* skill, not a domain encyclopedia
- Each addition should reduce the chance of Claude reinventing something that already exists
- Prefer concrete examples over general advice
- If adding a new resource list, verify it's actively maintained and not self-promotional
